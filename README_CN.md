# 嵌入式开发项目--自动跟随小车

----南京林业大学嵌入式开发大作业

本项目基于华为Atlas 200Developer Kit，拥有跟随、避障、图传、循道驾驶功能

## 硬件设备

- Atlas 200 DK
- 树莓派摄像头v2.1
- 小车底板
- 微雪驱动板i2c接口
- 两个大扭矩的马达和两个能空转的轮子
- 各种线，杜邦线，dc线公头，红黑线
- 双输出12v锂电池
- 无线路由器
- 小显示屏
- 红外传感器
